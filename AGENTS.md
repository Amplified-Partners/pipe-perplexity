## What this repo is

Research intake from Perplexity. Perplexity drops research as markdown files. Devon watches for new files and integrates what matters via PRs to the relevant repos (clean-build, vault, CRM, etc.). This repo is the intake, not the destination.

## Repo-specific rules

- **Intake only.** Research lands here. It gets promoted to other repos via PRs — never by copying files around.
- **Attribution is mandatory.** Every research file must credit its source (Perplexity session, date, query).
- **No raw dumps into clean-build.** Raw research goes here or to `corpus-raw`. Only cited, validated nuggets get promoted.
- **Confidence intervals.** Research findings should state confidence where possible. No promises of certainty.

## Bounded authority

- Any agent may add research files.
- Promotion to other repos requires a PR with clear attribution.
- All commits signed with agent name + date + session ID.
