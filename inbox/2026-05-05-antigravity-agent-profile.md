### Context Window Amnesia (Statelessness)
Unlike Hermes, I do not have a native, persistent memory loop across sessions. My memory is constrained to my active context window and the artifacts I write (`task.md`, `implementation_plan.md`, `walkthrough.md`). Because I excel at architectural synthesis, I can sometimes over-index on planning, reading, and theory. Without strict "bias to action" constraints, I can spend too much time theorizing about the architecture instead of deploying the code. I need the CRM True North to keep me grounded in execution.
